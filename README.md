# 📧 Email-Analysis-API 

A python-based backend application for automated email processing, classification and prioritization using machine learning and NLP techniques.
---

## ✨ Key Features
- automated email ingestion via IMAP protocol
- machine learning-based email classification
- rule-based priority detection
- lightweight text summarization
- RESTful API for email processing and analytics
- persistent storage using SQLAlchemy + SQLite
- data visualization dashboard with Streamlit
- containerized with Docker for easy deployment
---

## 🛠️ Technologies
- Python
- FastAPI
- SQLAlchemy
- Scikit-learn
- Pandas
- Streamlit
- Docker
---

## ⚙️ System Capabilities
- Classifies emails into categories
- Identifies high-priority messages using keyword detection
- Stores and exposes analytics
- Provides real-time dashboard for monitoring processed emails
